# Head Caricature Studio v3

זרימה מלאה:
1. מסך פתיחה — העלה ראש / צלם עכשיו
2. מסך מצלמה — ספירה לאחור 3, 2, 1 וצילום אוטומטי
3. מסך חיתוך ראש — תצוגת הראש כולל השיער
4. מסך בחירת סגנון — 2 אופציות: עדין / מודגש
5. מסך תוצאה — בחירה, ייצוא PNG, או הפעלה מחדש

## מה יש בגרסה הזאת
- חיתוך ראש + שיער בדפדפן באמצעות MediaPipe
- שתי וריאציות קריקטוריסטיות לבחירה בכל פעם מחדש
- PWA בסיסי לעבודה נוחה על iPad
- תיקיית backend אופציונלית אם תרצה להעביר את הסגנונות לעיבוד שרת בעתיד

## חשוב
כרגע שתי אופציות הקריקטורה רצות ישירות בדפדפן ומבוססות על stylization גרפי מתקדם, לא על מודל AI חיצוני.
הקוד מוכן להמשך שדרוג לשרת AI נפרד, אבל כדי לעשות "קריקטורה AI מלאה" תצטרך לחבר backend עם מודל ייעודי.

## הרצה מקומית
```bash
python3 -m http.server 8080
```
ואז לפתוח:
```text
http://localhost:8080
```

## העלאה ל-GitHub Pages
תעלה את הקבצים בשורש הריפו, ואז:
- Settings
- Pages
- Deploy from branch
- main / root

## backend אופציונלי
```bash
cd backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

אפשר בהמשך לחבר את ה-frontend ל-API הזה במקום לרנדר את הסגנון ישירות בדפדפן.
